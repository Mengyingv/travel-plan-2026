## 行程概览

### 🏔️ 班夫国家公园 — 4天（7月中旬）
- 西雅图 → 卡尔加里 → 自驾班夫
- Lake Louise、Moraine Lake、冰原大道、Johnston Canyon
- Fairmont Chateau Lake Louise 下午茶
- 酒店对比：Chateau Lake Louise / Banff Springs / Otter Hotel / Hotel Canoe
- 预估费用：$650–950/人（经济方案）| $1,800–2,700/3晚（Fairmont 方案）

### 🇨🇳 中国 — 14天（8月底–9月中）
- 西雅图 → 首尔转机（3.5h）→ 北京
- 北京（7天自行安排）→ 青岛 → 重庆 → 成都 → 北京 → 首尔（16h停留）→ 西雅图
- 机票：$739 往返（韩亚+大韩航空）
- 预估费用：$985–1,480/人（北京段自理）
