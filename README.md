# 🌏 2026 夏季旅行计划

## 行程概览

### 🏔️ 班夫国家公园 — 4天
- 西雅图 → 卡尔加里 → 自驾班夫
- Lake Louise、Moraine Lake、冰原大道、Johnston Canyon
- 预估费用：$650–950/人

### 🇨🇳 中国 — 14天
- 西雅图 → 首尔转机 → 北京
- 北京（7天自行安排）→ 青岛 → 重庆 → 成都 → 北京 → 首尔 → 西雅图
- 预估费用：$985–1,480/人（北京段自理）
